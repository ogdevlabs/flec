## Pre-PDLC baseline — 2026-07-07 to 2026-07-08

### Added
- Exploration mode — narrates shapes and colors from the camera feed
- Challenge mode — voice-commanded find-it games with match/celebration feedback
- Reading mode — fingertip tracking across a page, reads the nearest word aloud
- Story mode — autonomous picture-book read-aloud with illustration description (BLIP-2)
- Wake word ("Hey Flec") + voice command parsing (start/cancel/repeat challenge, shutdown)
- Wear detection that gates all modes when the mask is removed
- Low-light detection from mean pixel brightness
- AR developer overlay (companion screen) drawing detection boxes and fingertip trails
- Queue-decoupled session loop with `ResponseEngine` as the single audio/AR gatekeeper
- Model downloader (`scripts/download_models.py`) and camera lister (`scripts/list_cameras.py`)
- Test suite: 206 tests across contract, integration, and unit (hypothesis property tests + perf benchmarks)
